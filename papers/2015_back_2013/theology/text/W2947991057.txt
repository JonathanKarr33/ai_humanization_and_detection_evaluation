DeÔ¨Ānition 14.6. A function f : K ‚ÜíK is called:

14.3 Sierpi¬īnski-Zygmund functions
In this section we will consider a type of functions that Ô¨Ārstly appeared
in 1920‚Äôs (see [38]). The classical Luzin‚Äôs Theorem implies that for every
Lebesgue measurable function f : R ‚ÜíR there is a set S ‚äÜR with inÔ¨Ānite
measure such that f|S is continuous. In 1922 (see [19]) H. Blumberg showed
that if we omit the assumption that f : R ‚ÜíR is measurable, then some weaker
version of the assertion of the Luzin‚Äôs Theorem remains true. More precisely
H. Blumberg proved the following.
Theorem 14.7 ([19]). Let f : R ‚ÜíR be an arbitrary function, then there exists
a dense subset S ‚äÜR such that f|S is continuous.
In the proof the set S was countable so naturally one can ask if the set S can
be uncountable. The partial answer to this was given by W. Sierpi¬īnski and A.
Zygmund in [38].
Theorem 14.8 ([38]). There exists a function f : R ‚ÜíR such that for any set
Z ‚äÜR of cardinality c the restriction f|Z is not a Borel map.
Obviously under (CH) the restriction of this function to any uncountable set
cannot be continuous. J. Shinoda (see [37]) proved that with (MA) + ¬¨(CH)
for every function f : R ‚ÜíR there exists an uncountable set Z ‚äÜR such that
f|Z is continuous. On the other hand Gruenhage proved that there is a model
of ZFC in which c > ŌČ1 and there is a function f such that f|Z is not Borel
for any uncountable set Z ‚äÜR (for details see [36]). By classical theorems of
Luzin and Nikodym, a function from Theorem 14.8 is nonmeasurable and does
not have Baire property. Moreover, although it is possible to construct it to be
injective, it is nowhere monotone in the sense that its restriction to any set of
cardinality c is not monotone. Let us introduce the following notion.
DeÔ¨Ānition 14.9. We say that a function f : K ‚ÜíK is Sierpi¬īnski-Zygmund
function if for every set A ‚äÜK of cardinality c, the restriction f|A is not a
Borel map.
